# Wall of Fame Challenge

Welcome to the CodeHarborHub Git Challenge! 

### Goal: 
Submit a Pull Request (PR) to add your name to the contributors list.

### How to contribute:
1. **Fork** this repository.
2. **Clone** your fork to your computer.
3. **Create a branch**: `git checkout -b add-yourname`.
4. **Edit** `contributors.md`: Add your name to the list.
5. **Commit & Push**: `git push origin add-yourname`.
6. **Open a Pull Request** on GitHub.

Happy Coding!
